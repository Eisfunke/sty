Exporting / Copying¶

as_dict method¶

Each register-object has an as_dict method. With it you can convert your register-object into a dict.

from sty import fg as fg_obj

fg = fg_obj.as_dict()

a = fg['red'] + "I have a red fg." + fg['rs']

print(a)

as_namedtuple method¶

Each register-object has an as_namedtuple method. With it you can convert your register-object into a namedtuple.

from sty import fg as fg_obj

fg = fg_obj.as_namedtuple()

a = fg.yellow + "I have a yellow fg." + fg.rs

print(a)

copy method¶

Each register-object has a copy method. With it you can make a copy of it.

from sty import fg as fg_a, RgbFg

fg_b = fg_a
fg_c = fg_a.copy()

fg_a.set_style('orange', RgbFg(240, 100, 100))

a = fg_a.orange + "I have an orange fg." + fg.rs
b = fg_b.orange + "I have an orange fg too." + fg.rs

print(a, b, sep="\n")

try:
    msg = "But I have no orange fg, because I'm an earlier copy."
    fg_c.orange + msg + fg.rs
except AttributeError:
    print(msg)
